Project Overview

The Challenge

Design and develop a comprehensive council services website that serves multiple user groups including residents, businesses, community members, and travelers. The website needed to provide easy access to council services, environmental information, community resources, and governance details while maintaining accessibility and user-friendly navigation.

The Solution

I created an 8-page website with intuitive navigation, featuring dedicated sections for different user needs. The site uses a consistent blue color scheme (#165870 and #84DAF1) with professional imagery and follows accessibility guidelines. Each page was designed with specific user requirements in mind, from council tax information to environmental initiatives.

Key Features Implemented

  • 8-page navigation structure with clear user pathways
  • Search functionality for quick information access
  • Responsive design for multiple device compatibility
  • Accessibility features including proper contrast and semantic markup
  • Section-specific color coding for intuitive navigation
  • Interactive elements and hover effects for enhanced UX

Project Reflection

üéØ What I Learned

This project enhanced my understanding of user-centered design principles and the importance of designing for multiple user groups simultaneously. I learned to balance functional requirements with visual appeal while maintaining accessibility standards throughout the development process.

üöÄ Challenges Overcome

The main challenge was maintaining design consistency across different sections while making each area unique for its specific user group. I successfully balanced uniformity with section-specific requirements, such as the green theme for environmental content and categorical organization for community services.

üí° Design Evolution

The progression from Task 2 wireframe to Task 3 implementation demonstrates significant growth in my design capabilities. I evolved from basic site mapping to functional navigation with active page indicators, hover effects, and professional visual cohesion using the established color palette.

üîÆ Future Improvements

Priority improvements include replacing the aerial photo with a properly scaled version, implementing scroll functionality for Community & Wellbeing navigation, comprehensive mobile testing, interactive feedback forms, and enhanced search bar functionality with dropdown suggestions.
